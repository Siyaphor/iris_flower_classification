# iris_flower_classification


## Description
This project focuses on classifying iris flowers into three species:
Setosa, Versicolor, and Virginica using Machine Learning.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Approach
- Loaded the Iris dataset
- Performed basic data analysis and visualization
- Trained a Logistic Regression model
- Evaluated model accuracy

## Result
The model achieved high accuracy in classifying iris flower species.
